发表于 2019-05-31 | 分类于 LeetCode

  • 难度：中等
  • 分类：数组
  • 解决方案：双指针

今天我们学习第18题四数之和，这是一道中等题。像这样数组的题目经常作为面试题来考察面试者算法能力和写代码能力，因此最好能手写出该题。下面我们看看这道题的题目描述。

题目描述

给定一个包含n个整数的数组nums和一个目标值target，判断nums 中是否存在四个元素a，b，c和d ，使得a + b + c + d的值与 target相等？找出所有满足条件且不重复的四元组。

注意：答案中不可以包含重复的四元组。
